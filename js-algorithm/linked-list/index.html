# 链表-linkedList

链表是什么？

  • 多个元素组成的列表
  • 元素存储不连续，使用 next 指针连在一起

数组和链表大的区别

数组：增删非首尾元素时往往需要移动元素。

链表：增删非首尾元素，不需要移动元素，只需要更改 next 的指针

技术要点：

链表里的元素存储不是连续的，之间通过 next 连接

JavaScript 中没有链表，但是可以用 Object 模拟链表

链表常用操作：修改 next、遍历链表。
